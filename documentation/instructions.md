## Quick Start

1. Clone or download this repository.
2. Open `index.html` in your browser (or serve it with a local dev server, e.g. VS Code Live Server).
3. Fill out the student form and click **Add Student** — the record appears in the table below and is saved automatically.
4. Refresh the page — your data is still there, loaded from `localStorage`.

No installation, no dependencies, no database setup required.
