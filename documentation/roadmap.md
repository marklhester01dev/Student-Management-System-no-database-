## Development Roadmap

This project is built in increments:

### Increment 1: Project Setup, JS Web Components, User Interface

- HTML/CSS/JS scaffolding
- Component architecture (custom elements)
- Form + table built as components

### Increment 2: CRUD Operations

- Create, Read, Update, Delete for student records

### Increment 3: Searching, Sorting, Filtering Students

- Search by ID/name/course
- Sort by ID/name/course/year level
- Filter by course/year level/gender

### Increment 4: Local Storage Persistence and Dashboard Statistics

- Save/load records via localStorage
- Live dashboard counts and breakdowns

### Increment 5: Form Validation

- Required fields, email/contact format, duplicate-ID checks

### Increment 6: UX Polish

- Responsive design, messages, confirmations
